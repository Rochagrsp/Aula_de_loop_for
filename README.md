# Análise de Vendas e Bônus em Python

Este notebook contém uma série de exercícios e exemplos práticos em Python para demonstrar:

*   Utilização de loops FOR para iterar sobre listas.
*   Implementação de condicionais IF e ELSE para lógica de negócios.
*   Cálculo de bônus e contagem de eventos baseados em metas.
*   Identificação de valores máximos em listas.

--- 

## Seções do Notebook

### Atividade 01: Cálculo de Bônus para Funcionários

Este código calcula o bônus para uma lista de vendas de funcionários. Se a venda atingir ou superar uma meta definida, um percentual de bônus é aplicado. Caso contrário, o bônus é zero.

### Atividade 02: Contagem de Dias Acima da Meta

Esta seção conta quantos dias de vendas diárias excederam uma meta pré-definida. Utiliza um loop FOR e uma condição IF para incrementar um contador.

### Atividade 03: Bônus Fixo por Vendedor

Calcula um bônus para uma lista de vendas de vendedores. Se a venda atingir ou superar uma meta, um bônus fixo (calculado com base na meta e um percentual) é atribuído. Caso contrário, o bônus é zero.

### Atividade 04: Quantidade de Vendas que Atingiram a Meta

Conta quantas vendas em uma lista de vendas atingiram ou superaram uma meta específica.

### Atividade 05: Cálculo de Bônus com Condição Extra

Este código calcula o bônus para cada venda. Além do bônus percentual por atingir a meta, um bônus extra é adicionado para vendas de alto valor (iguais ou superiores a R$2000).

### Atividade 06: Identificando a Maior Venda

Este código percorre uma lista de vendas para encontrar e exibir o maior valor de venda registrado na lista.
